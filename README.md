#ANIME_ZONE

Discover and explore your favorite anime on this comprehensive website, featuring a user-friendly search bar, detailed information with trailers, character galleries, and a curated selection of upcoming, airing, and popular anime on the homepage.
This project utilized React's Context API for efficient data propagation within the component tree while employing styled-components for styling. Data fetching was accomplished using the Jikan. moe API and Material UI was employed for incorporating icons.
